Programación

Equipo

Equipo que usan nuestros estudiantes

Computadora

Contamos con computadoras aptas para el aprendizaje en programación, Todas cuentan con varios programas para acceder a diagramas de flujo, editores de código, computadores virtuales, etc. Estas computadoras además cuentan con conexión a internet.

Las computadoras reciben mantenimiento de forma continua para que se encuentren en excelente estado para el uso que les darán los alumnos.

Impresora 3D

Contamos con impresoras 3D en las que alumnos de años superiores podrán acceder para trabajar y experimentar con impresión 3D, aprendiendo a utilizarlas de forma correcta y aplicándolo en un uso pedagógico.

Laptops

Aparte de las computadoras de escritorio, contamos con un equipo que administra el empleo de multiples laptops para prestar a los alumnos en caso de que no alcancen la cantidad de computadores de escritorio. Estas laptops cuentan con varios programas, exactamente los mismos programas que las computadoras de escritorio, con el propósito de que todos accedan a estas herramientas. Todos los alumnos pueden pedir prestadas estas laptops y luego devolverlas en cierto horario. Estas laptops cuentan con conexión a internet.

Perfil del egresado

¿Que vas a aprender?

Creación de páginas web

Aprenderás a diseñar y desarrollar sitios web modernos, interactivos y adaptables para distintos dispositivos utilizando tecnologías actuales.

Desarrollo de aplicaciones móviles

Podrás crear aplicaciones funcionales para celulares y tablets, enfocadas en la experiencia del usuario y la resolución de necesidades reales.

Programación de sistemas

Aprenderás a desarrollar software capaz de automatizar tareas, organizar información y resolver problemas de manera eficiente.

Desarrollo backend y bases de datos

Trabajarás con sistemas que almacenan, administran y procesan información utilizada por aplicaciones y páginas web.

Automatización de procesos

Crearás programas y herramientas que permiten realizar tareas repetitivas de forma automática, optimizando tiempo y recursos.

Diseño de videojuegos

Aprenderás todas las herramientas y sistemas que se necesitan para la creación de videojuegos
